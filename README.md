# Moment 2 - Frontend

Syftet med den här uppgiften var att skapa en applikation som agerar som en att göra-lista. I applikationen finns ett formulär där användare kan lägga till nya inlägg i listan och listan visas även på startsidan. Inläggen läggs till med titel, en valfri beskrivning och en status om inlägget är avklarat, pågående eller inte startat. För varje inlägg kan man även ändra status eller radera inlägg. 

Applikationen är skapad med React och Vite och är skriven med TypeScript. Delar av webbplatsen är uppdelad i komponenter, exempelvis formulär, listan och knappar för att radera och uppdatera. Detta för att skapa en bra struktur. CSS-filer är också skapade för varje enskild komponent.
